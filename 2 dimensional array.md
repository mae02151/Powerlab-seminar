# 2차원 배열 생성

 ### 2차원 배열 정의하는 법
 - 세미콜론을 사용하여 정의
 - 행과 행사이에 엔터를 입력하여 정의
 - 매트랩에서 제공하는 함수 이용하여 정의

### 세미콜론을 이용하여 2차원 벡터 생성하기
```
>> A = [2, 9, -1 ; 4, 27, 19]

A =
       2  9 -1
       4  27  19
```
### 행과 행사이에 엔터를 입력하여 벡터 생성하기
```
>>B = [7 1
-4 8
0 47]

B = 
      7   1
      -4  8
      0   47
```
